# 📦 Logistics Cost Calculator  
### 🚀 Project Goal:
A lightweight calculator for delivery costs based on weight and distance — designed to simulate a logistics pricing system.

### 🔧 Features:
- Calculates delivery costs based on weight and distance
- Handles invalid weights/distances
- Supports easy customization for pricing and weight rules

### 🛠️ Technologies:
- Python (input handling, conditionals, calculations)

### 📈 Future Improvements:
- ✅ Dynamic pricing based on weight ranges  
- ✅ Delivery time estimator  
- ✅ Data logging for cost analysis  

### 👨‍💻 How to Run:
```bash
# Clone this repo
git clone https://github.com/yourusername/logistics-cost-calculator

# Run the script
python logistics_cost_calculator.py
